# Cara Install

### Install Nodejs (root only)
- Install nvm (node version manager) di armbian linux
- letakan file ini di file manager armbian desktop cth : /Documents/requester
- ```nvm install 16.5```
- ```nvm use 16.5```
- ```node -v```
- ```npm install pm2 -g```
- ```pm2 save```

### Install Firefox
- ```sudo apt install firefox```
